📋 What happens next?

1Clone & Setup

The installer downloads CypherPulse to your home directory and creates a Python virtual environment with all dependencies.

2Configure API

Get your free API key from twitterapi.io and add it to your .env file along with your Twitter username.

3Start Dashboard

Run cypherpulse serve to launch the web dashboard at http://localhost:8080 and start tracking your tweet performance!
